# Gold-Price-Prediction
Machine Learning project for predicting gold prices using financial market indicators and regression models such as Random Forest and XGBoost. The project includes data preprocessing, exploratory data analysis, feature engineering, model evaluation, and interactive visualizations for accurate price forecasting.

# 📌 Project Overview

This project focuses on predicting gold prices using Machine Learning algorithms based on historical financial market data.  
The project includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation.

Multiple regression models were implemented and compared to identify the best-performing model for gold price forecasting.

# 📂 Dataset Information

The dataset contains historical financial indicators affecting gold prices, including:

- SPX (S&P 500 Index)
- USO (United States Oil Fund)
- SLV (Silver Prices)
- EUR/USD Exchange Rate

---

# 🤖 Machine Learning Models Used

The following regression models were trained and evaluated:

## 1️⃣ Linear Regression
Used as a baseline model for predicting continuous gold prices.

## 2️⃣ Decision Tree Regressor
Captured non-linear relationships and complex decision patterns in the dataset.

## 3️⃣ Random Forest Regressor
An ensemble learning model that improved prediction accuracy and reduced overfitting.
